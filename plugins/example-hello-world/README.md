# Hello World Plugin

Example plugin for Ren'IDE demonstrating the Plugin API.

## Features

- **Project Statistics**: Displays counts of blocks, characters, variables, and labels
- **Settings Persistence**: Counter that persists across IDE sessions
- **API Testing**: Buttons to test toast notifications, logger, and dialog APIs
- **Plugin Information**: Shows plugin metadata and available APIs

## Installation

1. Build the plugin:
   ```bash
   cd plugins/example-hello-world
   npm install
   npm run build
   ```

2. Copy the plugin to your user data directory:
   ```bash
   # macOS/Linux
   cp -r plugins/example-hello-world ~/.vangard-ide/plugins/

   # Windows
   xcopy plugins\example-hello-world %APPDATA%\.vangard-ide\plugins\ /E /I
   ```

3. Restart Ren'IDE

4. The "Hello World" tab should appear in the plugin menu

## Development

To build in watch mode:
```bash
npm run dev
```

## What This Plugin Demonstrates

- **Plugin Factory Pattern**: Default export function that receives `PluginContext`
- **State Access**: Read-only access to blocks, characters, variables, labels, and analysis results
- **Logger API**: Prefixed console logging (`info`, `warn`, `error`)
- **Toast API**: UI notifications with different severity levels
- **Dialog API**: Modal dialogs for messages
- **Settings API**: Persistent key-value storage namespaced per plugin
- **React Components**: Full React support with Tailwind CSS classes
- **Error Handling**: Automatic error boundary wrapping (try throwing an error to see it)

## API Reference

### PluginContext

```typescript
interface PluginContext {
  state: {
    blocks: Block[];
    analysisResult: RenpyAnalysisResult;
    projectImages: Map<string, ProjectImage>;
    projectAudios: Map<string, RenpyAudio>;
    characters: Map<string, Character>;
    variables: Variable[];
    projectRootPath: string | null;
  };
  api: {
    logger: {
      info(message: string, ...args: any[]): void;
      warn(message: string, ...args: any[]): void;
      error(message: string, ...args: any[]): void;
    };
    toast(message: string, type?: 'success' | 'error' | 'warning' | 'info'): void;
    dialog: {
      showMessage(title: string, message: string): Promise<void>;
      showError(title: string, message: string): Promise<void>;
    };
  };
  settings: {
    get(key: string, defaultValue?: any): any;
    set(key: string, value: any): Promise<void>;
  };
  plugin: {
    id: string;
    version: string;
  };
}
```

### Plugin Instance

```typescript
interface PluginInstance {
  onDestroy?: () => void;
  tabs?: Record<string, React.ComponentType>;
}
```

## Next Steps

To create your own plugin, copy this example and modify:
1. Update `manifest.json` with your plugin's metadata
2. Implement your plugin logic in `src/index.tsx`
3. Add additional tabs as needed
4. Build and deploy to `~/.vangard-ide/plugins/your-plugin-id/`
